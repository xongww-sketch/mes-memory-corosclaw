### 二、税务架构核心概念解析

#### 1. 为什么需要复杂的跨境架构？

**直接出口的劣势**：
- 国内直接卖给国外客户，利润留在国内，需交 25% 企业所得税
- 通过香港中转，利用离岸利润豁免，香港利得税可降至 0

**架构价值**：
- 利润沉淀在香港（低税区），而非国内（25% 企业所得税）
- 广东高驰出口享受 13% 增值税退税
- 香港离岸利润可申请利得税豁免

#### 2. 关键税务概念

| 概念 | 解释 |
|------|------|
| **完税价格** | 海关计征关税的基准价格（CIF/FOB等） |
| **VAT（增值税）** | 国内 13%，出口可退税 |
| **出口退税** | 广东高驰采购原料付了13% VAT，出口后退还，降低实际成本 |
| **香港利得税** | 16.5%，但离岸利润可申请豁免 → 实际 0 |
| **离岸利润** | 香港公司从境外采购、销往境外，利润来源不在香港，可申请豁免 |
| **关联交易** | 高驰↔工厂（同集团内不同法人实体之间的交易） |
| **交易货币 vs 本位币** | 交易货币是合同币种，本位币是公司记账币种，差额记汇兑损益 |
| **税码映射表** | 主体×业务×国家→税码→税率→生效日期，税码不常变，税率和生效日期需定期维护 |

### 三、香港公司税务深度解析

#### 离岸利润豁免的实质

- 香港公司从日本工厂采购成品 → 直接销往各国 → **属于离岸贸易**
- 购销两端都在境外，利润来源不在香港 → 利得税豁免 → **实际税负 0**
- 但需满足：合同签署、资金收付、物流安排均在境外完成

#### 利润回流路径

- 利润留在香港 → 通过分红回广东高驰 → 内地对香港分红预提税 5%（税收安排优惠）
- 若不回流：国内运营成本通过广东高驰自身利润（加工费/贸易差价）覆盖
- **高驰心存高远有限公司**（24年成立的香港持股公司）：可能是红筹架构的顶层控股平台

#### 直接出口 vs 经香港中转

- 直接出口：利润留在国内，交 25% 企业所得税
- 经香港：利润沉淀在香港，离岸豁免 0 税，分红回国 5% 预提税
- **结论**：经香港中转综合税负更低

### 四、SAP 主数据驱动 + 规则引擎

#### 设计思想

**主数据**：客户、供应商、物料、税码等基础数据，全系统共享，一处维护处处生效。

**规则引擎**：基于主数据自动判断业务规则，无需硬编码。

**典型案例**：
- 税码映射：主体×业务×国家 → 自动匹配税率
- 客户信用额度：超额度自动拦截销售订单
- 价格确定：客户组+物料组+数量 → 自动定价

#### 税码映射表的变动逻辑

- **税码**：不常变（如"中国标准增值税"这个分类不变）
- **税率**：会变（如增值税从 16% 降到 13%），需维护生效日期
- **维护方式**：一般需人工跟进政策变化，更新到系统，暂无全自动逻辑

### 五、企业架构与资本运作

#### BVI/开曼公司

- **BVI（英属维尔京群岛）/开曼**：离岸金融中心，零税/低税
- **作用**：控股平台、避税、海外融资便利、上市主体

#### 红筹架构

- **定义**：境内资产通过境外控股公司实现境外上市
- **标准结构**：创始人 → BVI → 开曼（上市主体）→ 香港（中间层）→ 境内运营公司
- **香港作为中间层**：内地对香港分红预提税 5%（vs 其他地区 10%），有税收安排优惠

#### 利润回流与输血

- 香港→国内分红：5% 预提税
- 香港→国内借款：需付利息，利息部分需交税
- 国内自负盈亏：广东高驰通过加工费/贸易差价维持运营
