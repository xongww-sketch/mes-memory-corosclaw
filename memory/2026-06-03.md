## 📊 MES→coros后台 同步逻辑全景（彻底理清）

coros后台不直读MES生产库，读中转 schema `mes_daily_data`（1主表+3明细）：

```
daily_info(主表)
  ├─ production_id → production_detail（生产数据）
  ├─ delivery_id   → delivery_detail（含channel）
  └─ pack_id       → pack_detail（包装数据）
```

三条定时喂数任务：
- **get_mesdailydata**（每天1点）：源 mes.machine_data(procedure_detail_id=58等)+joborder → 写 production_detail+daily_info
- **get_mespackingdata**（每天1点）：包装工序 → 写 pack_detail+daily_info
- **get_channel**（每天3点）：国内mes_delivery_order_details+国外OMS snoutbounds → 写 delivery_detail+daily_info（channel在此步进来）

coros后台拉取入口：mes_info（webhook POST /mes_info，带sn查）→ daily_info LEFT JOIN 三明细

**渠道问题根治链**：管易同步 → 灌mes_delivery_order_details → get_channel → daily_info → coros后台
