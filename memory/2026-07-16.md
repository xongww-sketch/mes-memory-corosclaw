### 主题一：贸易模式与出口路径

**B2B vs 一般贸易出口 vs 跨境电商海关编码**
- B2B = 公司间买卖（企业对企业），是交易模式
- 9610（跨境电商 B2C 直邮）/ 9710（B2B 直接出口）/ 9810（B2B 出口海外仓）是海关监管代码，不是一般贸易
- 一般贸易出口 = 传统 B2B 先卖给境外公司，再由对方上平台零售
- 广东直接出口海外：走香港 vs 不走香港的核心区别在于**利润留存地和税务安排**

**经过香港的最大目的**
- 把利润留在香港公司（香港利得税 16.5%，远低于内地企业所得税 25%）
- 广东高驰卖给香港高驰 → 香港高驰卖给海外经销商/国外高驰公司
- 属于**离岸贸易**，广东高驰只确认卖给香港公司的利润

### 主题二：SAP 主数据体系

**三大核心主数据 + 其他**
- 客户主数据、供应商主数据、物料主数据是三大核心
- 还有：财务主数据（总账科目/成本中心/利润中心）、资产主数据、BOM、工艺路线、工作中心等

**客户 vs 供应商主数据维护差异**
- 客户主数据：销售视图（价格清单、付款条件、统驭科目）→ 决定"怎么卖、怎么收钱"
- 供应商主数据：采购视图（采购组织、采购价格清单、付款条件）→ 决定"怎么买、怎么付钱"
- 两者通过**统驭科目**关联到财务总账，实现业务→财务自动过账

**SAP "配置+主数据驱动" 设计思想**
- 核心思想：**业务规则不写死在代码里，而是通过配置表+主数据组合实现**
- 条件技术（Condition Technique）：价格/税码/折扣等通过"条件表+存取顺序+条件类型"动态匹配
- 举例：同一个物料卖给不同客户价格不同 → 客户主数据里的价格清单 + 物料主数据里的定价 → 系统自动匹配
- 税码映射同理：国家+物料类型+客户类型 → 自动带出正确税率

### 主题三：公司架构与股权设计

**香港心存高远 100% 控股广东高驰的意图**
- **税务优化**：广东高驰利润通过分红给香港心存高远，香港对境外所得不征税
- **资本运作**：香港公司方便对接资本市场（港股/海外融资），VIE 架构常见
- **风险隔离**：内地经营风险不直接传导到香港主体
- 广东高驰仍需交企业所得税，但分红给香港可通过税收协定降低预提税

**心存高远 vs 广东高驰 vs 香港高驰的关系构想**
- 心存高远（香港）：控股平台 + 资本运作主体
- 广东高驰：生产制造主体（内地工厂）
- 香港高驰：贸易主体（离岸销售、利润中心）
- 典型架构：生产利润留在广东 → 贸易利润留在香港 → 资本收益通过心存高远

### 主题四：越南/日本代工厂逻辑

**为什么国内产能够用还要找越南和日本代工**
- 不是产能问题，是**地缘政治+关税+供应链多元化**
- 越南：规避中美关税（美国对中国加征关税，越南出口美国税率更低）
- 日本：贴近日本市场，本地化生产提升品牌信任度
- 本质：高驰不自己造，而是"买佳禾的成品来卖"，用采购/销售单据链串货权和账务

**转让定价规则**
- 要求关联交易按"独立第三方公平价格"（Arm's Length Principle）
- 广东高驰卖给香港高驰的价格，必须接近卖给无关联第三方的价格
- 否则税务局有权调整补税

**增值税全链路**
- 抵扣进项税的目的：避免重复征税，只对"增值部分"交税
- 举例：广东高驰买原料 100 元（进项税 13 元）→ 卖给香港高驰 150 元（销项税 19.5 元）→ 实际交税 = 19.5 - 13 = 6.5 元
- 出口环节：出口免增值税（零税率），进项税可退税

### 主题五：其他概念

**同股不同权（AB 股）**
- A 股：1 股 1 票（公众股东）
- B 股：1 股多票（创始人/管理层，通常 10:1 或 20:1）
- 目的：创始人用少量股权保持控制权，同时大量融资
- 港股/美股允许，A 股不允许

**价格清单到底是什么**
- 不是"一张纸"，是 SAP 里的**定价条件记录**
- 定义：某个客户/客户组 + 某个物料 + 某个时间段 = 什么价格
- 销售渠道和价格清单的关系：不同销售渠道（直销/经销/电商）对应不同价格策略
- 供应商的采购价格清单同理：定义从哪个供应商买什么物料、什么价格
